Arty's Web Dev Cheatsheet

Listen in HTML

HTML-Tag Name Beschreibung
<ul> Unordered List UngeordneteListe in Stichpunkten
<ol> Ordered List GeordneteListe mit Nummerierung
<li> List Item Einzelne Punkte der Listen
<dl> Description List Liste mit Über- und Unterpunkten
<dt> Description Term Überschriften für nachfolgende Punkte einer Definitionsliste
<dd> DTerm Description Unterpunkte in einer Definitionsliste

Tabellen in HTML

HTML-Tag Name Beschreibung
<table> Table Hauptcontainer einer Tabelle
<thead> Table Head Seperater Tabellenkopf
<caption> Table Caption Überschrift einer Tabelle
<th> Table Heading Leitet eine Kopfzelle ein
<tbody> Table Body Eigentlicher Tabellenkörper
<tr> Table Row leitet eine neue Tabellenreihe ein
<td> Table Datacell Leitet eine neue Tabellenzelle ein
<tfoot> Table Footer Fußzeile einer Tabelle
